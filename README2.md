# Student List FASTAPI

## Features
- Show Students
- Add Student
- Edit Student
- Delete Student

## Tech Stack
-Python
- FastAPI

## Description
This project is a simple student management application built using FastAPI.  
Users can:
- View student list
- Add new students
- Edit existing student details
- Delete Student 

## Installation

```bash
uvicorn  main:app --reload
```

## Project Structure

```bash
src/
 ├── models/
 ├── routers/
 ├── database.py
 └── main.py
```
