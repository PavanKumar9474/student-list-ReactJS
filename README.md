# Student List ReactJS

## Features
- Show Students
- Add Student
- Edit Student
- Delete Student

## Tech Stack
- React.js
- JavaScript
- CSS

## Description
This project is a simple student management application built using React.js.  
Users can:
- View student list
- Add new students
- Edit existing student details

## Installation

```bash
npm install
npm run dev
```

## Project Structure

```bash
src/
 ├── components/
 ├── pages/
 ├── App.js
 └── main.js
```
